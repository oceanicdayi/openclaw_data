# 彈性理論教學內容

## 1. 基本概念

### 1.1 彈性力學簡介
彈性力學是研究固體材料在外力作用下產生變形，並在去除外力後恢復原狀的科學。主要應用於地震學中的地殼變形分析。

### 1.2 應力與應變
- **應力 (Stress)**: 單位面積上的內力
- **應變 (Strain)**: 變形程度的度量
- **胡克定律**: σ = Eε (線性彈性範圍)

## 2. 彈性常數

### 2.1 主要彈性參數
- **楊氏模數 (Young's Modulus, E)**: 衡量材料抵抗拉伸或壓縮變形的能力
- **剪切模數 (Shear Modulus, G)**: 衡量材料抵抗剪切變形的能力
- **體積模數 (Bulk Modulus, K)**: 衡量材料抵抗體積變化的能力
- **蒲松比 (Poisson's Ratio, ν)**: 橫向收縮與縱向伸長的比值

### 2.2 關係式
```
G = E / [2(1 + ν)]
K = E / [3(1 - 2ν)]
```

## 3. 彈性波動方程

### 3.1 波動方程推導
從牛頓第二定律出發，結合應力-應變關係，推導出彈性波動方程：

```
ρ ∂²u/∂t² = (λ + 2μ)∇(∇·u) - μ∇×(∇×u)
```

其中：
- ρ: 密度
- u: 位移向量
- λ, μ: 拉梅常數

### 3.2 地震波類型
- **P波 (縱波)**: 壓縮波，傳播速度最快
- **S波 (橫波)**: 剪切波，傳播速度較慢

## 4. 地殼彈性特性

### 4.1 典型岩石彈性參數
| 岩石類型 | 楊氏模數 (GPa) | 蒲松比 | 密度 (kg/m³) |
|---------|---------------|--------|-------------|
| 花崗岩  | 50-70         | 0.2-0.3 | 2600-2800   |
| 玄武岩  | 60-80         | 0.25-0.3 | 2800-3000   |
| 砂岩    | 10-30         | 0.2-0.3 | 2000-2600   |
| 石灰岩  | 20-70         | 0.2-0.3 | 2400-2700   |

### 4.2 地殼分層彈性特性
- **上地殼**: 較脆性，彈性模數較低
- **下地殼**: 較韌性，彈性模數較高
- **莫霍面**: 彈性性質顯著變化

## 5. 應用於地震預警

### 5.1 彈性波傳播模擬
利用彈性理論建立地震波傳播模型，預測地震波到達時間和強度。

### 5.2 斷層力學分析
- 應力累積與釋放機制
- 斷層滑移預測
- 餘震序列模擬

### 5.3 地震早期預警參數
- P波初動特徵
- 震級快速估算
- 震源機制解

## 6. 數值模擬方法

### 6.1 有限元素法 (FEM)
- 複雜幾形狀處理
- 非均質材料模擬
- 邊界條件靈活設定

### 6.2 有限差分法 (FDM)
- 波動方程直接求解
- 計算效率高
- 適合大規模模擬

### 6.3 譜元素法 (SEM)
- 高精度波動模擬
- 減少數值耗散
- 適合地震波傳播
